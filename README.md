# Gizli Sohbet Aktivasyon Paneli
Bu proje, e-posta veya telefon numarası gerektirmeyen, tek seferlik ücretli giriş sistemine sahip bir sohbet sitesinin yönetim ve doğrulama merkezidir.

## Klasör Yapısı
- `/admin`: Kod üretme ve dışa aktarma araçları.
- `/backend`: API ve doğrulama mantığı.
- `/data`: Veritabanı dosyaları (Gitmeyecek şekilde .gitignore'a ekle).
